# 2026-07-08 — Major Architecture Refactor

## Summary

Today we did a ground-up architectural refactor of the entire codebase. The goal was to make the framework fully generic — any robot, any controller, any estimator, any trajectory generator, any physics engine — all config-driven with zero hardcoded wiring.

## What Changed

### PhysicsEngine ABC (`physics_engine/`)
- Extracted `PhysicsEngine` abstract base class from `components.py` into `physics_engine/base.py`-adjacent
- `MuJoCoEngine` now implements `PhysicsEngine` — name-based joint/body access, no hardcoded LeKiwi slices
- `ArmRobot` no longer imports `mujoco` — talks to the engine through the protocol

### Registry Pattern (`factories/registry.py`)
- Four registries: `_CONTROLLER_REGISTRY`, `_ESTIMATOR_REGISTRY`, `_TRAJECTORY_REGISTRY`, `_PLANT_REGISTRY`
- Each concrete class registers itself with a decorator (`@register_controller("LQR")`, `@register_plant("ArmRobot")`, etc.)
- Factories are now 3-line generics: lookup `from_config` → instantiate
- Adding a new component: just add the decorator + `from_config` + TOML config — no factory code changes

### TOML-Driven Configuration
- All configs moved from YAML to TOML (built-in `tomllib`, no dependency)
- `robot_config.toml` — defines the robot model, joint groups, plants
- `configs/controllers/*.toml` — controller type + weights
- `configs/estimators/*.toml` — estimator type + gains
- `configs/trajectories/*.toml` — trajectory type + waypoints

### New Modules
- `simulation/robotsim.py` — `RobotSim` generic factory (reads TOML, creates engine + plants via registry)
- `factories/` — registry and three factories
- `demos/` — four demo scripts (simple, arm, base tracking, pick-and-place)
- `deprecated/` — old demo scripts moved as-is
- `scripts/generate_robot_config.py` — auto-generates `robot_config.toml` from MJCF XML

### Key Files Moved
- `RobotSim` from `lekiwi_sim.py` → `simulation/robotsim.py`
- `MuJoCoEngine` from `lekiwi_sim.py` → `physics_engine/mujoco.py`
- `PhysicsEngine` ABC from `components.py` → `physics_engine/__init__.py` (re-exported)

### Removed (Dead Code)
- `run_simulation()` — zero callers
- `MuJoCoEngine.get_drive_qpos()`, `set_arm_ctrl()`, `set_drive_ctrl()` — zero callers
- `HolonomicMobileRobot.set_pose()` — zero callers
- `lekiwi-sim/so_arm100.xml` — unused XML referencing nonexistent meshes
