Lekiwi MPC — System Architecture

Whole-Body Control Framework · MPC + Kinematics + Simulation

Architecture

  • • Three ABCs: Controller, Plant, StateEstimator
  • • Plants: HolonomicMobileRobot (3-DOF), ArmRobot (6-DOF)
  • • Controllers: PID, MPC_LTI, LQR
  • • Arm uses Cartesian state + IK-in-step pattern
  • • Controller never touches joint space

Arm Kinematics

  • • Forward kinematics: homogeneous transforms
  • • Geometric Jacobian (verified vs numerical)
  • • Inverse kinematics: damped pseudoinverse
  • • Step clamp prevents limit overshoot
  • • Round-trip verified: state → IK → FK → match

Controllers

  • • PID: joint-space position control, anti-windup
  • • MPC_LTI: OSQP QP solver, precomputed H/F
  • • LQR: DARE solve, gain_calculation separated
  • • All controllers inherit from same ABC
  • • Swap controller without changing plant
